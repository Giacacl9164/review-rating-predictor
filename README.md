## 🚀 Getting Started

review-rating-predictor is a Windows app that predicts product review ratings from review text. It uses TF-IDF and Naive Bayes behind a simple web interface built with Streamlit.

Use it to:
- Paste or type a product review
- Get a predicted star rating
- Test how the model reads review language

## 🧭 How to Use the App

1. Start the app
2. Open the page in your browser
3. Find the text box
4. Paste or type a product review
5. Click the predict button
6. Read the predicted rating

Example inputs:
- This product works very well and feels worth the price
- The item arrived late and stopped working after two days
- Good quality, easy to use, and I would buy it again

The app reads the words in the review and estimates the star rating based on patterns it learned from past data.

## 🧩 How the Model Works

The app uses a simple text model:
- TF-IDF turns words into numbers
- Naive Bayes looks for patterns in those numbers
- The model then predicts a review rating

This setup works well for text that has clear word patterns. It gives fast results and fits a small desktop app.
